# 이미지 번역 프로그램

## 프로그램 개요
평소 영어로 된 만화나 영상 등을 자주 즐기는 편이라, 모르는 단어나 문장이 나왔을 때 그 이미지 자체를 번역해주는 기능이 필요했습니다. 그래서 파파고 이미지 번역이나 구글 이미지 번역 등을 자주 사용했고, 이에 흥미가 생겨 이미지 번역을 할 수 있는 프로그램을 만들어보고자 했습니다.

## 실행 과정
1. 이미지 업로드 - 드래그&드롭 또는 Ctrl+V 붙여넣기
2. OCR (문자 인식) - EasyOCR을 통해 이미지 속 텍스트 추출
3. 번역 - deep-translator를 이용해 영어 텍스트를 한국어로 번역
4. 원문 영역 블러 처리 - OpenCV로 원문 텍스트 영역 흐리게 처리
5. 번역문 표시 - Pillow로 번역문을 원래 위치에 맞춰 표시
6. 결과 저장 및 출력 - OCR/번역 결과를 `.txt` 파일로 저장




위 과정을 통해, 번역하고자 하는 이미지를 업로드하면, 기존 이미지 위에 번역한 텍스트를 얹은번역된 이미지를 보여주고, 추출한 텍스트의 결과를 txt형태로 저장하는 프로그램을 만들어 보았습니다.

가끔 이미지 인식을 제대로 하지 못하는 경우가 있어, 우선 test.png로 한 번 해보시길 추천드립니다.
